Stage 1: beginner (0-6 months)

Goal:Build tech foundations, basic security awareness, and hands-on lab skils


Skills To Master:

  1. Computer Literacy:

    • Typing,file management,Os Basics (Windows/Linux)
    • Duration:2 weeks
     

  2. Newtworking Fundamentals

    • IP addresses, DNS, HTTP/S, routers, firewalls, TCP/UDP.
    • Duration:1 month
     

  3. Linux CLI

    • Commands (ls, cd, grep), file permissions, package management.
    • Duration:1 month
     

  4. Programming Basics (Python)

    • Variables, loops, functions, simple scripts (e.g., port scanner).
    • Duration:2 month
     

  5. Security Concepts

    • CIA triad, malware, phishing, password hygiene.
    • Duration: 2 weeks
